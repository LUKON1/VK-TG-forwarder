# VK → TG Forwarder

Пересылает сообщения и фотографии из VK бесед в Telegram-группы.  
Поддерживает несколько маршрутов: одна беседа VK → один чат TG.

## Как работает

VK Community Bot добавляется в беседы как участник с правом читать все сообщения. Бот слушает события через **Bots Long Poll API** и пересылает их в TG через Bot API.

```
VK Беседа(ы) → Bots Long Poll → роутер → Telegram Bot API → TG Группа(ы)
```

**Формат сообщений в TG:** `[Имя Фамилия]: текст`

## Требования

- Node.js >= 20
- Docker + Docker Compose (для деплоя)
- VK-сообщество (любого типа)
- Telegram-бот от @BotFather

## Настройка VK-сообщества (один раз)

**1. Включить сообщения и бота**  
Управление → Сообщения → включить  
Управление → Сообщения → Настройки для бота → **"Разрешать добавлять сообщество в беседы"**

**2. Получить токен**  
Управление → Дополнительно → Работа с API → Создать ключ → право: **Сообщения**

**3. Включить Long Poll**  
Управление → Дополнительно → Работа с API → Long Poll API → включить

**4. Добавить сообщество в беседу**  
На странице сообщества → "Добавить в чат" → выбрать беседу

**5. Дать права на чтение всех сообщений**  
В беседе → Участники → сообщество → **Доступ ко всей переписке**  
*(делает любой администратор беседы)*

Повторить шаги 4–5 для каждой беседы.

## Конфигурация

Скопируй `.env.example` → `.env` и заполни:

```env
VK_COMMUNITY_TOKEN=  # токен из шага 2
TG_BOT_TOKEN=        # от @BotFather

# Маршруты: vk = peer_id беседы, tg = ID Telegram-группы
ROUTES=[{"vk":2000000007,"tg":-1001234567890}]

# Опционально: Прокси (HTTP / HTTPS / SOCKS)
# HTTPS_PROXY=socks5://100.123.173.100:1080
```

Если вашему боту нужен доступ через прокси, раскомментируйте и установите переменную `HTTPS_PROXY` (или `HTTP_PROXY`). Бот автоматически применит её для всех запросов к Telegram и VK.

**Как узнать peer_id беседы:** запусти бота, напиши в беседу — в консоли появится:
```
[peer] incoming=2000000007 match=false
```
Добавь эту пару в ROUTES и перезапусти.

**Несколько маршрутов:**
```env
ROUTES=[{"vk":2000000007,"tg":-1001234567890},{"vk":2000000008,"tg":-1009876543210}]
```

## Запуск

### Локально
```bash
npm install
npm start
```

### Docker (на сервере)
```bash
docker compose up --build -d
```

Контейнер автоматически поднимается при перезагрузке (`restart: unless-stopped`).

## Когда нужно вмешаться

Только если токен сообщества был вручную отозван в панели управления.  
В остальном — работает автономно без ограничений по времени.
